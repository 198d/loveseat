Loveseat by 198d

Simple Couch Client / Object Document Mapper

Small, lightweight CouchDB client and object document mapper. Library is meant to be kept lean while still exposing the most powerful features of the database. Took a little inspiration from sunspot.

Goals

  • Let Couch do the work

    Do as little as possible in actual library. Make Couch handle all of the heavy lifting.

  • Stay out of the way

    Avoid as much as possible interfering with other libraries allowing choice amongst things like validations, callbacks, etc...

  • Treat design documents as first-class citizens

    Expose the design document at the same level as a normal document allowing for full exposure to the design document API.

Dependencies

json (1.4.6)

Install

$ gem install loveseat
